What is HTML

HTML is Hyper Text Markup Language is a mark up language that add tags to text to define:

  • structure
  • meaning

What is CSS

CSS for short for the cascading style sheet, it controls the presentation of the webpage:

  1. layout
  2. colour of text
  3. colour of background
  4. size of HTML elements

In summary HTML manages the structure, content and words whereas CSS manages the colours, styles and sizes.
